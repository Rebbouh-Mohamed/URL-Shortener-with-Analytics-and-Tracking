#  URL Shortener with Analytics & Tracking

A lightweight, self-hosted link shortener with built-in analytics and click tracking. Shorten URLs, monitor performance, and gain insights with real-time dashboards — all while keeping full control of your data.

##  Features
- **URL Shortening** – Create short, clean links instantly
- **Click Analytics** – Track total clicks, unique visitors, and trends
- **Visitor Insights** – View geographic, device, and browser breakdowns
- **Custom Domains** – Use your own branded domain
- **QR Codes** – Generate QR codes for shortened links
- **API Ready** – Full REST API for programmatic access
- **Lightweight** – Minimal dependencies, easy to deploy

Perfect for marketers, developers, and anyone needing to track link performance without third-party analytics.
